Proyek Pilihan 🚀

Diet-Assistance (Distance)

Diet Assistance (Distance) adalah aplikasi web berbasis AI yang dapat mengenali makanan dari gambar, menghitung jumlah kalori per 100 gram, dan memberikan rekomendasi diet secara otomatis. Sistem ini dikembangkan menggunakan Flask, TensorFlow/Keras, dan Nutritionix API, serta dilengkapi fitur autentikasi pengguna dan riwayat prediksi.

Lihat di GitHub

Diabetes-Prediction (DiaPredict)

Diabetes-Prediction (DiaPredict) adalah aplikasi web yang menggunakan model berbasis AI untuk memprediksi risiko diabetes berdasarkan data-data yang dibutuhkan seperti usia, hipertensi, riwayat penyakit jantung, IMT, kadar glukosa darah, kadar HbA1c, riwayat merokok, dan jenis kelamin. Sistem ini dikembangkan menggunakan metode Random Forest dan XGBoost.

Lihat di GitHub

Website E-Commerce (Online Shop)

Proyek ini merupakan pembuatan dan pengembangan website e-commerce seperti Online Shop sederhana yang berfungsi sebagai platform untuk menampilkan, mengelola, dan melakukan transaksi produk secara online. Website ini dibuat menggunakan teknologi berbasis website seperti HTML, CSS dan JavaScript.

Lihat di GitHub

Stroke Prediction

Proyek ini bertujuan untuk membangun model prediksi risiko stroke menggunakan teknik machine learning. Dalam implementasinya, dua algoritma pembelajaran mesin digunakan untuk dibandingkan, yaitu Random Forest dan Naive Bayes.
